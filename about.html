How to update the index

Place all your PM&R HTML topic files in: C:\Users\Fahri\Documents\School Work\PM&R\SEM 8

  • Copy build_pmr_hub.py into that folder.
  • Double click run_build_hub.bat, or run python build_pmr_hub.py from Command Prompt.
  • The script will create or refresh index.html, about.html, and the assets folder.
  • Open index.html in your browser. You can pin or bookmark it.
  • Whenever you add, rename, or delete topic HTML files, run the builder again.

Suggested folder structure

You can keep all files in one folder, or organize them into subfolders. Both approaches work.

  • SEM 8\Stroke\stroke_revision.html
  • SEM 8\Cardiac Rehab\cabg_rehab.html
  • SEM 8\Musculoskeletal\knee_acl_oa.html
  • SEM 8\index.html

Important note

A normal offline browser cannot automatically scan a Windows folder by itself for security reasons. That is why this hub uses a local Python builder script to create the topic list.
