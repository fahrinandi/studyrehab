PM&R Offline Study Hub
======================

Purpose
-------
A simple interactive webpage in preparation for the PM&R national board exams.

Target folder
-------------
C:\Users\Fahri\Documents\School Work\PM&R\SEM 8

Installation
------------
1. Extract this ZIP.
2. Copy these files/folders into:
   C:\Users\Fahri\Documents\School Work\PM&R\SEM 8

   - build_pmr_hub.py
   - run_build_hub.bat
   - index.html
   - about.html
   - assets folder

3. Double click run_build_hub.bat.
4. Open index.html in your browser.

How to update
-------------
Whenever you add, rename, move, or delete HTML topic files, run run_build_hub.bat again.

Why a builder script is needed
------------------------------
For security reasons, an offline browser cannot scan your Windows folder automatically.
The Python script scans your folder locally and writes assets/topics.js.
Nothing is uploaded.

Notes
-----
- Existing topic HTML files are not edited.
- index.html and about.html are regenerated by the builder.
- The script scans subfolders too.
